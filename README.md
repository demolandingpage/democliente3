# 🍷 Corte alle Ferrazze — Landing Page
https://demolandingpage.github.io/democliente3/
Landing page ufficiale dell'Osteria Corte alle Ferrazze, ristorante tradizionale veneto a Scimmia (VR).

## 🚀 Come usare

1. Clona la repository
2. Apri `index.html` nel browser
3. Carica le tue immagini nella cartella `img/` rispettando la struttura

## 📁 Struttura immagini

- `img/hero/` → Sfondo hero (1920x1080)
- `img/piatti/` → Card menu (800x600)
- `img/eventi/` → Foto eventi
- `img/ambiente/` → Locali e cantina
- `img/staff/` → Chef e team

## 🛠️ Tecnologie

- HTML5 semantico
- Tailwind CSS (CDN)
- Google Fonts (Playfair Display, Lora)
